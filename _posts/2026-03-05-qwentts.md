**零基础小白也能玩转Qwen3-TTS！阿里开源AI语音神器超详细新手教程**


今天这篇教程专门写给**完全不会电脑技术**的朋友——哪怕你连“终端”是什么都不知道，只要跟着我一步一步抄代码、点鼠标，就能让电脑用AI帮你**生成超级自然的中文/英文语音**，还能**克隆自己的声音**、**按描述定制声音**！

Qwen3-TTS 是阿里Qwen团队2026年1月刚开源的顶级TTS模型，支持10种语言、低延迟流式生成、语音设计、3秒快速克隆，效果吊打很多商业工具，而且完全免费！

### 第一步：最简单的方法——不用装任何东西，直接网页试用

不想折腾电脑？先去玩在线Demo，5秒就能听到效果：

- Hugging Face Demo（国际）：https://huggingface.co/spaces/Qwen/Qwen3-TTS  
- ModelScope Demo（国内更快）：https://modelscope.cn/studios/Qwen/Qwen3-TTS  

打开网页 → 输入文字 → 选声音/描述 → 点击生成 → 直接听！  
玩够了再来装本地版（本地版速度更快、可以无限用、不怕网络）。

### 第二步：本地安装（小白专属傻瓜式教程）

#### 1. 准备一个干净的Python环境（5分钟）

我们用 **Miniconda**（最适合小白的工具），它不会破坏你电脑原来的程序。

1. 打开浏览器，访问 https://docs.conda.io/en/latest/miniconda.html  
2. 下载适合你系统的版本（Windows选.exe，Mac选.pkg）  
3. 双击安装 → 一直点“Next”或“Install”就好（默认选项）  

安装完后：

- **Windows用户**：按 Win + S 搜索 “Anaconda Prompt”，打开它  
- **Mac用户**：打开 “Terminal”

#### 2. 创建专用环境（复制粘贴就行）

在打开的黑色窗口里，**逐行复制下面命令**，每行按回车：

```bash
conda create -n qwen3-tts python=3.12 -y
conda activate qwen3-tts
```

看到 `(qwen3-tts)` 就成功了！

#### 3. 安装Qwen3-TTS工具包

继续在同一个窗口输入：

```bash
pip install -U qwen-tts
```

如果你的电脑有NVIDIA显卡（大部分游戏本都有），再装加速包，速度会快很多：

```bash
pip install -U flash-attn --no-build-isolation
```

（如果报错或卡住，加一句 `MAX_JOBS=4` 放在最前面也行）

安装完就大功告成啦！模型下载会自动进行，第一次可能要等几分钟（1.7B模型大概4-6GB）。

### 第三步：开始生成语音！（复制代码就行）

#### 方法一：用预设好听的声音说话（推荐新手从这里开始）

新建一个文本文件，命名为 `tts_test.py`，用记事本或任意编辑器打开，**全部复制下面代码**进去：

```python
import torch
import soundfile as sf
from qwen_tts import Qwen3TTSModel

# 加载模型（第一次会自动下载）
model = Qwen3TTSModel.from_pretrained(
    "Qwen/Qwen3-TTS-12Hz-1.7B-CustomVoice",  # 推荐这个模型
    device_map="auto",           # 有显卡自动用GPU，没显卡用CPU
    dtype=torch.bfloat16,
)

# 生成语音（改这里就行！）
wavs, sr = model.generate_custom_voice(
    text="大家好，今天教大家用AI生成语音超级简单！",
    language="Chinese",          # 可以改成 English
    speaker="Vivian",            # 可选声音：Vivian, Serena, Uncle_Fu, Dylan, Ryan 等
    instruct="用特别兴奋的语气，语速稍快一点"   # 可加描述，不想加就删掉这行
)

# 保存成wav文件
sf.write("我的第一个AI语音.wav", wavs[0], sr)
print("生成完成！文件叫 我的第一个AI语音.wav")
```

保存后，在终端输入：

```bash
python tts_test.py
```

几秒后就会在文件夹里出现音频文件！双击就能听。

**支持的声音列表**（直接抄名字用）：
- Vivian（明亮年轻女声，中文最自然）
- Serena（温柔女声）
- Uncle_Fu（稳重大叔）
- Dylan（北京口音男生）
- Eric（四川口音男生）
- Ryan / Aiden（英文男声）
- Ono_Anna（日语女生）
- Sohee（韩语女生）

#### 方法二：语音设计（你描述，我生成声音）

把上面的代码改成下面这个（用 **VoiceDesign** 模型）：

```python
model = Qwen3TTSModel.from_pretrained("Qwen/Qwen3-TTS-12Hz-1.7B-VoiceDesign", device_map="auto", dtype=torch.bfloat16)

wavs, sr = model.generate_voice_design(
    text="今天天气真好，我们去公园玩吧！",
    language="Chinese",
    instruct="用一个5岁可爱小女孩的声音，奶声奶气、超级开心地说"
)

sf.write("小女孩声音.wav", wavs[0], sr)
```

描述越详细越好！比如“低沉磁性大叔，带点沙哑，像在讲鬼故事”都可以。

#### 方法三：克隆自己的声音（3秒音频就够）

这次用 **Base** 模型：

```python
model = Qwen3TTSModel.from_pretrained("Qwen/Qwen3-TTS-12Hz-1.7B-Base", device_map="auto", dtype=torch.bfloat16)

wavs, sr = model.generate_voice_clone(
    text="你好，我现在用自己的声音说话啦！",
    language="Chinese",
    ref_audio="我的录音.wav",     # 把你的3-10秒录音放这里（wav/mp3都行）
    ref_text="你好。"   # 录音里说的文字（可不写，模型会自动识别）
)

sf.write("克隆我的声音.wav", wavs[0], sr)
```

**进阶玩法**：先生成一个完美声音，再反复克隆（保持同一声音）：

先用VoiceDesign生成一段参考音频 → 用 `create_voice_clone_prompt` 做成提示 → 后面每次生成都用这个提示，声音就永远一样！

### 第四步：更方便的网页界面（一键启动）

装完包后，在终端输入：

```bash
qwen-tts-demo Qwen/Qwen3-TTS-12Hz-1.7B-CustomVoice --ip 0.0.0.0 --port 7860
```

打开浏览器输入 `http://localhost:7860` 就能看到漂亮的网页界面！像用ChatGPT一样操作，再也不用敲代码了。

### 小贴士（避免踩坑）

- 第一次运行会下载模型，建议用有线网络，耐心等5-10分钟  
- 显卡内存小于8GB建议用0.6B小模型（把1.7B改成0.6B）  
- 生成的音频自动保存为wav，双击就能用微信/抖音发送  
- 想商用？模型是Apache-2.0协议，完全可以商用
